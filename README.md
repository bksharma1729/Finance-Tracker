# Finance-Tracker
Personal Finance Tracker is an application for tracking daily transactions, managing budgets, and analyzing financial behavior. It provides clear insights through dashboards and reports to support smarter financial decisions.
FinTrack – Finance Tracker

FinTrack is a modern personal finance dashboard built with React.
It allows users to track income, expenses, and overall balance through a clean, fintech-inspired interface.

✨ Features ****

📈 Dashboard overview (Balance, Income, Expenses)

🧾 Add / Edit / Delete transactions

🔍 Search & filter transactions

🌗 Light & Dark mode support

📊 Financial overview gauge

⚡ Responsive and modern UI

🛠 Tech Stack *****

React.js

React Router

Context API / Hooks

Custom CSS Theme System
